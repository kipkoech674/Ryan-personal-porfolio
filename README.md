# Ryan Kipkoech - Portfolio Website

A world-class, modern portfolio website showcasing the skills and projects of Ryan Kipkoech, Software Engineer.

## Features

- **Modern Design**: Clean, minimalist design with premium feel and smooth animations
- **Dark/Light Mode**: Toggle between dark and light themes with persistent preference
- **Fully Responsive**: Mobile-first design that looks great on all devices
- **Smooth Animations**: Engaging micro-interactions and scroll-triggered animations
- **SEO Optimized**: Proper meta tags and semantic HTML for better search engine visibility
- **Performance Optimized**: Fast loading times and optimized assets

## Sections

1. **Hero Section**: Powerful introduction with call-to-action buttons
2. **About Me**: Story-driven section highlighting skills and passion
3. **Skills**: Categorized technical and soft skills with progress bars
4. **Projects**: Showcase of 6 high-quality projects with descriptions and tech stacks
5. **Experience**: Timeline of work experience and education
6. **Testimonials**: Client and collaborator testimonials
7. **Contact**: Contact form and social media links
8. **Footer**: Quick links and additional information

## Tech Stack

- **React 18**: Modern UI library
- **TypeScript**: Type-safe code
- **Tailwind CSS**: Utility-first CSS framework
- **Vite**: Fast build tool and dev server
- **Lucide React**: Beautiful icon library

## Customization Guide

### 1. Update Personal Information

**Hero Section** (`src/components/Hero.tsx`):
- Update social media links (GitHub, LinkedIn, Email)
- Customize the tagline and description
- Add your actual CV link for the Download CV button

**Contact Section** (`src/components/Contact.tsx`):
- Replace placeholder email: `ryan.kipkoech@example.com`
- Add your phone number
- Update social media links
- Consider integrating a form backend (FormSpree, EmailJS, or Supabase)

### 2. Add Your Real Projects

**Projects Section** (`src/components/Projects.tsx`):
- Replace the 6 placeholder projects with your actual projects
- Add real GitHub repository links
- Add live demo URLs
- Update project images (use high-quality screenshots)
- Describe the problems you solved and technologies used

### 3. Update Experience & Education

**Experience Section** (`src/components/Experience.tsx`):
- Add your real work experience, internships, and freelance work
- Update education details for Kirinyaga University
- Include specific achievements and metrics
- Add dates and organization names

### 4. Customize Skills

**Skills Section** (`src/components/Skills.tsx`):
- Adjust skill levels to reflect your actual proficiency
- Add or remove skills based on your expertise
- Update categories if needed

### 5. Get Real Testimonials

**Testimonials Section** (`src/components/Testimonials.tsx`):
- Reach out to professors, classmates, or freelance clients
- Ask for LinkedIn recommendations
- Replace placeholder testimonials with real ones
- Include actual photos and LinkedIn profiles

**Solution**: Build an AI chatbot using GPT-4 that provides:

- Weather-based planting recommendations

**Tech Stack**: React Native, Python/FastAPI, OpenAI API, Twilio, PostgreSQL

**Impact**: Directly addresses a real problem in Kenya, shows social impact
